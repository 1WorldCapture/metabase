# Code

Our themed wrapper around [Mantine Code](https://mantine.dev/core/code/).

## When to use Code

The Code component allows users to display code snippets. Users can also use the `Anchor` component within the code block, but should consider using the `Anchor` component outside  the code block instead. For displaying text, use the `Text` component.
